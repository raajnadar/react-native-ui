# react-native-ui

Initial repository scaffold.

## Requirements
- Node.js (LTS)
- pnpm

## Getting Started
1. Install dependencies: `pnpm install`
2. Run the example app (if configured): `pnpm --filter example start`

## Workspace
This repo uses a pnpm workspace for the example app and any packages added later.

## Turborepo
Common commands:
- `pnpm run build`
- `pnpm run dev`
- `pnpm run lint`
- `pnpm run test`
- `pnpm run clean`
